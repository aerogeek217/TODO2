# Components

Detail reference for `src/components/` (task, canvas, overlays, shared, layout, settings, taskboard) and `src/views/` helpers. Load when touching UI, canvas nodes, overlays, editors, or view-level grouping/bucketing.

Empty / undefined value short-circuits back to the unfiltered list (no narrowing). The spec round-trips onto the loaded def via `writeCurrentStateToDef` / `handleCreateNew` + hydrates back via `applyDefinition`; the value is transient (mirrors widget surfaces — it is *not* persisted on the def).
